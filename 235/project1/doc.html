Documentation page

1. Header Section

The top part of the page features three distinct areas stacked on top of each other:

  • The top section is an empty box.
  • The middle section contains the publication name "MYSTERIOUS NAME 1999" and "KINEMATIK PUBLISHING" in a smaller size.
  • The bottom section has the main headline "THE SENTINEL," which is the name of the newspaper.

2. Article Section

This section is divided into two columns:

  • The left column holds the main content of the article, which is made up of multiple paragraphs of placeholder text.
  • The right column includes an image (a group of people) along with a quote and some additional text.

3. Navigation Section

This area is present but currently empty, it can be removed later or used if needed.

4. Aside Section

Another area that is empty at the moment because it is being used to create the layout for the page.

5. Footer Section

At the bottom of the page, this section is empty.

In essence, the layout has a traditional newspaper style with a header, article section, and footer.

6. Notes Section

  • This layout is a basic newspaper format.

  • The article/main content section is divided into two columns, one for text and one for images and quotes.

  • The header contains the publication's name and the article's headline.

  • The footer, aside, and navigation sections are in place for layout purposes, but do not hold content.

  • It was difficult finding a coffee stain image that closely matched the original one, so I used one that would cover the left most part of the middle header box.

  • The website in general needs to be wider.

  • The background needs to move with the page.

  • Currently, the page resizes to a narrower version of the current one. The planned responsive design is to have the elements on the page stack vertically, to mimic an article being viewed on mobile.

  • The color of the page needs to more closely match the source material.

  • Some of the fonts found in the source material are very hard to replicate. Also, some text still needs fonts applied to it.

  • Font in the blockquote needs to be adjusted to be bold and thicker
